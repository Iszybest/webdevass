list of tags and their functions

  1. h-tag:
    this tag is used for headings.
    It has six sizes which can be specified
    using h1 to h6.
  2. a_tag:
    this tag is used to redirect user to another web page.
  3. p_tag:
    tis tag is used to create a paragraph.
  4. nav_tag:
    this tag is used to create a navigation.
  5. ul_tag:
    this tag is used to create an unordered list.
  6. ol_tag:
    this tag is used to create an ordered list.
  7. li_tag:
    this tag is used to define elements of a list.
  8. img_tag:
    this is called the image tag and it is used to attach an image.
